# DateType in JS

### Number 
#### Represent the numbers.
    e.g :
        var num1 = 40;
        let num2 = -90

### Sting 
#### Collection of text.
        -> singel quotes : 'text'
        -> double quotes : "text"
        -> backtick : `text`

    e.g : 
        var name = "name";
        let f_name = 'demo';
        const l_name = `kumar`;  
### Boolean
#### True(1) & False(0).
    e.g : 
        var condition = true;
        let istrue = false;
### Undefined :  
#### Something not defined yet but maybe defined later.
    e.g :
        var status = undefined; 
### NULL
#### It actually represent empty values.
    e.g :
        let a; /* undefined */
        let b = 10;
        b = null;
### Object 
#### If we have to some how store key : value pair we can use objects.
    e.g :
        {
            name : "demo",
            company : "xyz",
            position : "SDE-1"
        }

### Primitive & Non-Primitive
- Primitive : Types which are atomic in nature.

        e.g : 
            number
            string
            boolen
            undefined
            null

- Non-primitive : Types which are a composition of other types.

        e.g :
            Objects,
            Array etc.
